# CyberNexo Prisma

Mascota virtual evolutiva para navegador móvil.

## Estado actual

Esta versión deja el juego en una línea evolutiva compacta de **6 fases**:

1. Núcleo Semilla
2. Bytecría
3. Lynx de Paquetes
4. Protocazador
5. Centinela Kernel
6. Soberano Nexo

## Características

- Tres barras principales: Energía, Integridad y Sincronía.
- Las barras bajan con el tiempo.
- Si una barra llega a 0, el nexo falla y se pierde la mascota.
- Evolución por puntos de datos.
- Galería inferior ajustada a 6 evoluciones.
- Evoluciones futuras ocultas hasta desbloquearlas.
- Música procedural inspirada en estética gótica/PSOne.
- Al evolucionar, la música se desvanece, suena una explosión digital y aparece una transición de luz azul antes de cambiar de forma.
- App autocontenida en `index.html`, pensada para ejecutarse directamente desde un navegador móvil.

## Uso

Abre `index.html` en el navegador. En móvil, toca la pantalla una vez para activar el audio.

## Nota

La versión del repositorio es una versión ligera y ejecutable directamente desde GitHub, usando dibujo y audio procedural para evitar subir assets pesados. Las versiones de trabajo con imágenes y WAV generados se mantienen como artefactos descargables fuera del repo.
